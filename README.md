# 3D-Virtual-Scene-Modeling

### 1、准备

1）按照“实验1.1”文档配置opengl的运行环境(如果非visual studio 2015的话请自行配置环境，这一步很关键)<br>
2）把项目clone下来，解压压缩包后打开，双击“.sln”文件用visual studio 2015打开<br>
3）ctrl+F5运行<br>
4）具体交互操作请参考“说明书”<br>
5）具体实现原理请参考“报告书”及代码注释<br>
